# DIC Personal Page

This is a modern, responsive personal webpage for **Bing-Rong Tsai**.

## Features

- **Live Local Time**: Features a dynamic custom JavaScript clock.
- **Premium Aesthetics**: Utilizes a combination of glassmorphism (frosted glass effects) and deep, animated background gradients.
- **Dynamic Animations**: Subtle micro-animations including floating background orbs, hover effects, and a blinking digital clock colon.
- **Responsive Layout**: Designed to look clean and professional on desktop, tablet, and mobile devices.

## How to View

To view the page, simply download the repository and open the `index.html` file in any modern web browser.

## Technologies Used

- **HTML5** for structure
- **CSS3** (Vanilla) for styling, animations, and typography (Playfair Display & Inter from Google Fonts)
- **JavaScript** (Vanilla) for the real-time clock logic
